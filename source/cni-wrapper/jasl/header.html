cni-jasl

Building the libraries defined in the sub-directories

The purpose of each target defined in the Makefile for this directory is listed below. The files are written to or removed from their respective output directories as follows:
  • Object (.o) files - < sub-directory name >/.obj
  • Header (.h) files - $JASL_BASE/include/jasl/cni/< sub-directory name >
  • Library files and links - $JASL_BASE/lib/jasl

  • obj - compiles all of the source files in each sub-directory using gcc, placing the object (.o) files in the corresponding directory noted above.
  • hdr - copies the header files in each sub-directory to the corresponding directory noted above.
  • none or libs - executes the actions defined for the obj target and then uses the object (.o) files to generate shared and static libraries, as well as the necessary links, in the directory noted above. Upon completion, the clean target is used to remove the directories containing the object (.o) files.

  • clean - removes each of the directories noted above that are used to store the object (.o) files.
  • clean_hdr - removes each of the directories noted above that contain copies of the header (.h) files.
  • clean_libs - deletes the shared and static library files, as well as the associated links, in the directory noted above.

  • docs - generates an HTML version of the Makefile using source-highlight, placing it in $JASL_BASE/share/jasl/doc/source/cni-wrapper/jasl. This file, jasl.html, is also generated, and includes links to all of the HTML files generated by the same target, called from this one, in each sub-directory.
