swig

Building the wrapper libraries defined in the sub-directories

The purpose of each target defined in the Makefile for this directory is listed below. The output files are written to or removed from $JASL_BASE/lib/jasl/{scripting language name (lower case)} for the libraries or the sub-directories of $JASL_BASE/bin, associated with their scripting language below, for the module files. All other files (e.g. generated source and object (.o) files) remain in their respective sub-directory.

  • perl - perl-jasl
  • python - py-jasl

The following targets are used to build or clean all of the libraries and files referenced above, "passing through" to the sub-directories, where the actual execution takes place.

  • none or libs

  • clean_all

  • docs - generates an HTML version of each interface file and the Makefile using source-highlight, placing the output files in $JASL_BASE/share/jasl/doc/source/swig. This file, swig.html, is also generated, and includes links to all of the HTML files generated by the same target, called from this one, in each sub-directory.
