CniWrapper

Building the wrapper libraries and their associated modules

The purpose of each target defined in the Makefile for this directory is listed below. The output files are written to or removed from $JASL_BASE/lib/jasl/{scripting language name (lower case)} for the libraries or the sub-directories of $JASL_BASE/bin, associated with their scripting language below, for the module files.

  • perl - perljASL
  • python - pyjASL

The following targets are used to build and clean the libraries and module files necessary to access the cni-wrapper library using the scripting language name (perl or python) included in the target name.

  • {language name}_src - uses swig to generate the library source file (CniWrapper_wrap.cxx). If successful, it also creates the module file, placing it in the directory noted above.

  • {language name}_obj - executes the actions defined for the {language name}_src target and then uses gcc to compile the generated source file into an object (.o) file.

  • {language name}_lib - executes the actions defined for the {language name}_obj target and then uses the object (.o) file to generate a shared library (CniWrapper.so) in the directory noted above.

  • clean_src - deletes the library source file (CniWrapper_wrap.cxx).
  • clean_obj - deletes the library object file (CniWrapper_wrap.o).

  • clean_{language name} - executes the actions defined for the clean_{language name}_module and clean_{language name}_lib targets.

  • clean_{language name}_module - deletes the module file(s) from the directory noted above.
  • clean_{language name}_lib - deletes the library from the directory noted above.

    The following targets are used to build and clean all of the libraries and files referenced above, using combinations of the targets associated with them, and the indicated documentation.

  • none or libs - executes the actions defined for each {language name}_lib target, followed by the clean target.

  • clean_all - executes the actions defined for the clean and clean_libs targets.

  • clean - executes the actions defined for the clean_src and clean_obj targets.
  • clean_libs - executes the actions defined for each clean_{language name} target.

  • docs - generates an HTML version of each interface file and the Makefile using source-highlight, placing the output files in $JASL_BASE/share/jasl/doc/source/swig/CniWrapper. This file, CniWrapper.html, is also generated, and includes links to all of the HTML files.
