DiceRollWizard

Managing files

The purpose of each target defined in the Makefile for this directory is listed below. They are also referenced by the Makefile in the parent directory during recursive documentation build operations.

  • docs - generates an HTML version of each source file and the Makefile using source-highlight, placing the output files in $JASL_BASE/share/jasl/doc/source/py-jasl/DiceRollWizard. This file, DiceRollWizard.html, is also generated, and includes links to all of the HTML files.
