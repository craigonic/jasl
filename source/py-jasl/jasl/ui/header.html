jasl.ui

Managing the packages defined in the sub-directories

The purpose of each target defined in the Makefile for this directory is listed below. Note that the targets are also referenced by the Makefile in the parent directory during recursive cleans and when generating documentation. They also "pass through" to the sub-directories.

  • clean - removes the __pycache__ sub-directory.

  • docs - generates an HTML version of the Makefile using source-highlight, placing it in $JASL_BASE/share/jasl/doc/source/py-jasl/jasl/ui. This file, ui.html, is also generated, and includes links to all of the HTML files generated by the same target, called from this one, in each sub-directory.
