py-jasl

Managing the files in the sub-directories

The purpose of each target defined in the Makefile for this directory is listed below.

  • clean - removes the __pycache__ sub-directories within the jasl hierarchy.

    The output files for the following targets are placed in the indicated sub-directory of $JASL_BASE/share/jasl/doc

  • docs - generates an HTML version of the Makefile using source-highlight, placing it in source/py-jasl. This file, py-jasl.html, is also generated, and includes links to all of the HTML files generated by the same target, called from this one, in each sub-directory.
  • doxygen - generates class documentation for the Python files using doxygen, placing the output files in py-doxygen/html.
