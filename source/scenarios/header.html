scenarios

Installing these files

The purpose of each target defined in the Makefile for this directory is listed below.
  • none or all - copies the JSON files to $JASL_BASE/share/jasl/data/scenarios.

  • clean - removes the JSON files from the output directory.

  • docs - generates an HTML version of the Makefile and each of the JSON files using source-highlight, placing the output files in $JASL_BASE/share/jasl/doc/source/scenarios. This file, scenarios.html, is also generated, and includes links to all of the HTML files.
